# Personality Quiz Hub

AI-powered personality quiz platform inspired by BuzzFeed's delightful UX aesthetic. Built with React, TypeScript, Tailwind CSS, and Framer Motion for smooth animations.

## Quick Start

```bash
npm install
npm run dev
```

## Structure

```
src/
├── components/     # React components
├── hooks/          # Custom React hooks
├── types/          # TypeScript definitions
├── utils/          # Helper functions
├── styles/         # Global styles
├── pages/          # Route components
└── App.tsx         # Root component
```

## Tech Stack

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Build**: Vite
- **API**: Groq for AI recommendations
